1.** District Performance Dashboard**
     A responsive analytics dashboard built using HTML, CSS, and Chart.js.
     
2.** Features**
     KPI Summary Cards (Male, Female, Others)
     Course Progress (Stacked Bar Chart)
     Pass Percentage (Horizontal Bar with % inside)
     Assessment Completion (Donut Chart)
     Grade Breakdown (Pie Chart with %)
     District Ranking Chart
     Light & Dark Theme Support
     Responsive Layout
     
3.**  Technologies Used**
     HTML5
     CSS3
     JavaScript
     Chart.js
     Chartjs-plugin-datalabels
